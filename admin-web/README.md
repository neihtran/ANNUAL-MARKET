# Chợ Truyền Thống Admin Web

Admin Dashboard cho nền tảng marketplace Chợ Tươi Thông.

## Công nghệ
- Next.js 14
- TypeScript
- Tailwind CSS + Shadcn/ui
- TanStack Table
- Recharts
- Axios

## Cài đặt

```bash
cd admin-web
npm install
```

## Chạy

```bash
npm run dev
```

## Chức năng

- Dashboard với biểu đồ thống kê
- Quản lý sản phẩm
- Quản lý đơn hàng
- Quản lý người dùng
- Responsive design
- Dark/Light mode support

## API Configuration

Tạo file `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:3001/api/v1
```
